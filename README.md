# PrevGestão Comercial

Aplicativo estático, pronto para publicar, focado na operação comercial previdenciária.

## O que já vem pronto
- login local
- dashboard com KPIs
- contratos com busca, filtros e edição
- botões rápidos para preencher novos contratos
- relatórios gerenciais
- usuários
- configurações de layout, tema, cores e marca
- backup e restauração
- base histórica real carregada a partir do arquivo RELATORIO CONTRATOS.xlsx

## Login inicial
- usuário: `admin`
- senha: `admin123`

## Como publicar amanhã
Você pode subir esta pasta em qualquer hospedagem estática:
- Netlify
- Vercel
- Cloudflare Pages
- GitHub Pages
- servidor próprio

Basta publicar os arquivos desta pasta.

## Observação importante
A base histórica foi carregada para uso comercial e operacional. Informações sensíveis que não precisam entrar no sistema comercial, como senhas e alguns dados sigilosos, ficaram de fora da interface por segurança.
